# chronver

> The [chronological](https://chronver.org "Official ChronVer website") versioner



## Install

```bash
$ npm i chronver
```

## Usage

```js
// ES6
import chronver from "chronver";
```

```js
// ES5
const chronver = require("chronver");
```
